01. NAVIGATION 📂
pwd Where am I?
ls -la See hidden files
cd .. Go up one level
cd - Jump to last folder
02. FILE OPS 📄
touch [file] Create empty file
mkdir -p Make nested folders
cp -r Copy folder contents
mv Move or rename
03. SUPER SIGHT 🔍
grep "text" Find text in files
tail -f Watch logs live
cat Read file content
find . -name Locate files
04. SYSTEM POWER ⚡
htop Monitor resources
chmod +x Make script runnable
ps aux List all processes
alias Make custom shortcuts
