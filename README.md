## 🧠 Why This Repo?

Because most docs look like this:

```
Very long explanation...
Very long explanation...
Very long explanation...
```

But comics look like this:

```
Dev: why is git not tracking my file?

git add file.js

Git: ok 👍
```

Much easier.

## ✏️ Want to Add a Comic?

Create a folder:

```
topic-name/
   index.html
```

Example:

```
kubernetes/
   index.html
```

Then add your comic panels.
