📓 THE GIT DEVELOPER SPELLBOOK

The Panic Pocket

Save your messy code in a hidden drawer to switch branches fast.

git stash push -m "Save"

Use "git stash pop" to get it back.

The Time Eraser

Fix your last commit (typos, forgotten files) without a new message.

git commit --amend --no-edit

Perfect for "Final... final_v2" cleanup.

The Undo Button

Undo the last commit but KEEP all your code changes.

git reset --soft HEAD~1

Safety first! --hard will delete your code.

The Map Viewer

See the beautiful, branching subway map of your history.

git log --oneline --graph --all

The best way to see where you are.

The Picky Thief

Grab one specific commit from another branch and bring it here.

git cherry-pick [HASH]

No need to merge the whole branch!

The Bug Detective

Let Git find which commit broke the app (Binary Search).

git bisect start

Mark commits 'good' or 'bad' to find the culprit.

The Ghost Tracker

Find "lost" commits that you accidentally deleted.

git reflog

Git never actually deletes anything for 30 days.

The Super Alias

Create a shortcut. Just type "git tree" from now on.

git config --global alias.tree "log --oneline --graph --all"

Set it once, use it forever.
